# 首頁「捲動放映」概念總綱

> 2026-07-18 定稿。角落官網新首頁方向:**捲動 = 放映一部一鏡到底的旅程電影**——滾輪往下前進、往上倒帶,行經各國「章節節點」時浮出互動層(地圖+行程),繼續滾動又滑走。同事試看後拍板走 **C 路線(寫實電影+疊層互動)**。
> 試看頁:`https://www.cornertravel.com.tw/_archive/film-demo.html`(A/B/C/D 四版,C 為主線)。

---

## 一、體驗架構

```
進站(<1秒即開演)
 └ 品牌開場層:logo+「走進世界的角落」浮在第一幕上,一捲動就退場
    └ 電影前進:紙門拉開 → 楓紅庭園 → 穿楓林 → 鳥居隧道 → 藝妓…(各國接龍)
       └ 章節節點(如鳥居=日本):遮罩淡入,左=單國金線地圖+城市脈衝點,
          右=行程選項卡(可點,連行程頁);繼續滾動整層滑出、電影續行
          └ …下一國章節(泰國、埃及…),直到片尾接正常內容流(關於/流程/聯絡)
```

- 倒捲=整部倒帶,所有層的進出都由捲動進度驅動、雙向可逆。
- 章節地圖**不用世界地圖**(放大有幾何碎片、也多餘)——用首頁 hero 現成的單國剪影地圖(全精度+城市點),inline 進頁面零依賴。

## 二、技術引擎(已建成,`_archive/`)

| 檔案 | 內容 |
|---|---|
| `scroll-film-ui.html` | **C 主線**:圖片序列引擎+品牌開場+日本章節節點 |
| `scroll-film-ui-b.html` | D(B 路線+雙節點),備選 |
| `scroll-film-a/b.html` | 純電影版 A/B(供對照) |
| `film/frames/s*/` | WebP 影格(1280w q70,行動/現行) |
| `film/frames-hd/s*/` | **1920 原檔單次壓縮 HD 影格(未上線,正式串接時切換)** |

引擎要點:
- **圖片序列取代影片元素**——影片高頻 seek 在 macOS 會吐黑格閃爍,拆幀畫 canvas 物理上不可能閃(Apple 產品頁同款)。
- **梯度載入**:首幀到就開演(正式站實測 0.6~1.6 秒),影格按 1/8→1/4→1/2→全部 背景補密,draw 取最近已載幀。**官網不准出現讀取等待。**
- 影格產線:`ffmpeg`(關鍵幀密度 `-g 12` 供對照用 mp4)→ 拆幀 → ImageMagick 轉 WebP(此機 ffmpeg 無 libwebp)。
- 部署=push main(站點綁 `www.cornertravel.com.tw`,舊 github.io 網址已 404);**上線後必跑線上行為測試**(Playwright 對正式網址,量開演用 `domcontentloaded`,`load` 會等全部影格)。

## 三、素材產線:首尾幀接龍(Lovart/Kling)

**鐵律:一對首尾幀 = 一個鏡頭動作。** 差距超過一個動作,模型必偷懶溶接(硬變場景)→ **插中繼幀拆開**(先用圖生圖把中間狀態釘成照片,再兩段各 5 秒)。已兩次靠此救回(超車段、瞳孔段)。

- 接龍:上一段尾幀=下一段首幀,轉場天生無縫。
- 時長一律選最長(10 秒);段內恆速、不剪接;抽卡不改字,同提示詞重抽,三抽不中換戰術(中繼幀/換邊)。

### 提示詞規矩(血淚版)
1. **責任分配寫法**:明講「她什麼都不變(constant pace, NEVER turns)、變的只有鏡頭(Only the CAMERA moves)」+ 用環境視差(pillars sweep past with parallax)逼模型理解鏡頭在動。
2. **禁 `film grain`**(顆粒被烤進影片+壓縮=髒噪點)→ 一律 `clean, sharp, high detail`;底片感之後由網頁端疊乾淨顆粒。
3. **禁 `drone` 字樣**(會畫出無人機)→ 用 first-person / camera glides + `no drone visible`。
4. **人物規則**:背影/側面/不看鏡頭/中景(臉是最大破綻面);唯一破例=引路人「跟我來」的正臉邀請鏡頭。人物皮膚 `natural skin texture, no beauty filter`。
5. **節奏四拍**:莊嚴(慢)→ 吸氣(遮擋爆點)→ 爽點(加速揭幕)→ 敬畏(靜)。

### 轉場文法庫(遮擋物=換世界的布)
楓葉穿越、潑水 match cut(大象的水潑上鏡頭)、門口強光洗白、**瞳孔傳送門**(推進正臉→眼睛微距→瞳孔吞畫面→虹膜反光長成新世界)、B 線另有「多重宇宙蓋拆重組」(建築縮時蓋了又拆,鏡頭恆速=我們是時間主體)。原則:**每一國找一個「當地的東西」蓋鏡頭**(埃及=風沙/門光、郵輪=浪花、越南=燈籠群)。

### 畫質
Kling 天花板 1080p(無 4K 輸出)——夠用;主犯是交付管線降階,已備 1920 HD 影格待切換;更高需求用 Topaz 類工具對單支後製。
